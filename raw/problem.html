Encoding

All this sites are encoded in UTF-8. Please verify whether the browser uses this encoding.
The default font is set to Charis SIL (version 4.0.02, free download site)
It is suggested to install this Unicode font before browsing this web site.

JavaScript

This web-based interface makes use of JavaScript. Please enable this feature within your browser's settings.

Browser Support

The following browsers are supported:

  • Safari (2.0) [recommended],free download site
  • Mozilla (1.7)
  • Firefox (2.0) [recommended],free download site
  • Flock (0.7.7) [recommended],free download site
  • Netscape (8.1) [not recommended]
  • Opera 9.0 [not recommended]
  • MS Internet Explorer 7.0 [not recommended]
  • Other Internet Explorer (IE) based browsers [not recommended]

Note: Certain lower versions of MS Internet Explorer (6.0) in Chinese and other non-Latin based languages may not display some IPA symbols properly, so please avoid using these kinds of browsers to view the contents of this web site.
